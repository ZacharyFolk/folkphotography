# Assigning Categories to Images - Quick Guide

## 🎯 Why This Matters

After the code review fixes, **categories now work for images**! This enables:
- ✅ Hero image category selector
- ✅ Location Map category filter
- ✅ Category Gallery widget filtering
- ✅ Better image organization

**You need to assign categories to your images for widgets to work properly.**

---

## 📋 Quick Start (2 Methods)

### Method 1: Individual Images (Best for Few Images)

**Time:** ~30 seconds per image

1. Go to **Media → Library**
2. Click on an image
3. On the right side, find **"Categories"** box
4. Check the categories (e.g., "Hero Images", "Travel", "Astro")
5. Click **Update**

### Method 2: Bulk Edit (Best for Many Images)

**Time:** ~2 minutes for 50 images

1. Go to **Media → Library**
2. Switch to **List View** (top right)
3. Check boxes next to images
4. Select **Bulk Actions → Edit**
5. Click **Apply**
6. In the bulk edit panel, select **Categories**
7. Click **Update**

### Essential Categories:

Create these categories if they don't exist:

1. **Hero Images** - For homepage hero rotation
2. **Astro Photography** - For astronomy photos
3. **Wildlife & Birds** - For wildlife photos
4. **Street Photography** - For street scenes
5. **Travel** - For travel photos
6. **Landscape** - For landscape photos
7. **Portraits** - For people photos
8. **Film/Holga** - For film photography

### How to Create Categories:

1. **Posts → Categories**
2. Enter name (e.g., "Hero Images")
3. Click **Add New Category**
4. Repeat for others

## 🎨 Assignment Strategy

### For Hero Images:

**Goal:** 10-20 images in "Hero Images" category

1. Pick your best wide/landscape photos
2. High resolution (2560x1440 or larger)
3. Visually striking
4. Assign to "Hero Images" category

**Why:** These will rotate on your homepage

### For Widgets:

**Location Map Widget:**
- Assign categories to photos with GPS
- Example: "Travel" category for travel photos with GPS
- Helps visitors explore by location + category

**Category Gallery Widget:**
- Assign specific category to showcase
- Example: "Astro Photography" for astronomy showcase
- Creates focused galleries

### For Organization:

**Primary category** = main subject
- If portrait of bird → "Wildlife"
- If street scene → "Street Photography"
- If stars → "Astro Photography"

**Additional categories** = context
- Also add "Travel" if taken while traveling
- Also add "Film/Holga" if shot on film

## ⚡ Bulk Assignment Workflow

### Scenario: You just uploaded 50 travel photos

**Steps:**

1. **Media → Library → List View**
2. **Sort by:** Upload Date (newest first)
3. **Select:** All 50 images (check top box)
4. **Bulk Actions → Edit → Apply**
5. **Categories:** Check "Travel"
6. **Update**

**Time:** 2 minutes for 50 images ✅

### Scenario: Creating hero rotation from existing images

**Steps:**

1. **Media → Library → Grid View**
2. **Browse** your best images
3. **Click** each hero-worthy image
4. **Right sidebar:** Check "Hero Images"
5. **Update**
6. **Repeat** until you have 10-20

**Time:** 5-10 minutes ✅

## 🔧 Troubleshooting

### "I don't see Categories box in Media Library"

**Fix:**
1. Edit an image
2. Top right corner: Click **Screen Options**
3. Check ✅ **Categories**
4. Categories box should appear on right side

### "Hero category selector shows no images"

**Fix:**
1. Verify categories are assigned to images
2. Check **Media → Library**
3. Filter by **Hero Images** category
4. Should show at least 1 image
5. If empty: Assign some images to "Hero Images"

### "Location Map widget shows all images, not filtered"

**Fix:**
1. Edit widget settings
2. Select specific category
3. Make sure images with GPS have that category
4. Save widget

## 💡 Pro Tips

### Tip 1: Use Bulk Edit for Speed
Don't assign categories one-by-one. Select 10-20 similar images, bulk edit.

### Tip 2: Hero Images = Your Best
Only put your absolute best photos in "Hero Images" category. Quality > quantity.

### Tip 3: Multiple Categories = More Visibility
A travel portrait can be in: Travel, Portraits, and Hero Images. More categories = more places it appears.

### Tip 4: Create Categories as Needed
Don't create 50 categories upfront. Start with 5-8, add more as you upload.

### Tip 5: Review Uncategorized Weekly
Go to Media → Library → Filter: Uncategorized → Assign categories weekly.
