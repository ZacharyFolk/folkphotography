**You have thousands of images. They're scattered. You feel stuck.**

This guide gives you a **simple daily system** to organize your photography library without burning out.

## ⏰ The 15-Minute Daily System

### Every Day (Monday-Friday)

**Time: 15 minutes maximum**

#### Step 1: Pick Your Batch (2 minutes)
Choose ONE source folder/drive/batch:
- Today: Desktop/RandomPhotos folder
- Tomorrow: External drive D:/2015
- Next day: Scanned film batch

**Rule:** Same source all week = easier decisions

#### Step 2: Select 3-5 Images (3 minutes)
From that batch, pick 3-5 images using this criteria:

**Quick Selection Method:**
- Scroll through batch
- First 3-5 that make you smile = selected
- Don't overthink it
- Imperfect > perfect

**Pro Tip:** Use Lightroom/Bridge quick select (P key)

#### Step 3: Quick Edit/Export (5 minutes)
- Open in DxO or Photoshop
- Basic adjustments only (exposure, crop if needed)
- Export with metadata preserved (see IMAGE-METADATA-GUIDE.md)
- Save to: `C:/ReadyToUpload/`

#### Step 4: Upload to WordPress (5 minutes)
1. Media → Add New
2. Drag all 3-5 images
3. While uploading, have categories ready
4. Click first image → Add title → Pick ONE category → 2-3 tags
5. Repeat for other images
6. Done!

**Result:** 3-5 new images added daily = 15-25 per week = 60-100 per month

## 📅 The Weekly/Monthly System

### Weekly: Sunday Review (30 minutes)

**Every Sunday:**

#### Review Week's Uploads (10 min)
- Go to Media Library
- Filter: Last 7 days
- Quick scan - any need recategorizing?
- Note which categories are growing

#### Bulk Organize (15 min)
- Pick ONE folder from backlog
- Rename files in batch (see File Naming below)
- Move to appropriate year/category folder
- Don't upload yet - just organize

#### Plan Next Week (5 min)
- Which folder will you pull from Mon-Fri?
- Note any scanning needs
- Check scanner/drives working

### Monthly: First Sunday - Portfolio Curation (60 minutes)

**Once a Month:**

#### Create Portfolio Collection (40 min)
1. Posts → Portfolio → Add New
2. Title: "[Month] [Year] Highlights" or "[Theme]"
3. Review last month's uploads
4. Pick 10-15 best images
5. Add to portfolio gallery
6. Write short description
7. Publish

#### Stats Review (10 min)
- Check camera stats widget
- Note favorite gear/settings this month
- See which categories grew most

#### Plan Next Month (10 min)
- Which batches to tackle?
- Any scanning projects?
- Set realistic goal (e.g., "Upload 60 images")

```
│   │   ├── 2020-scans/
│   │   └── 2024-scans/
│   ├── 35mm/
│   └── Medium-Format/
├── To-Scan/
│   ├── Priority/
│   └── Someday/
└── ReadyToUpload/  ← Your daily exports go here
```

### File Naming Conventions

**For Daily Uploads:**
```
Format: YYYY-MM-DD_category_description_###.jpg

Examples:
2024-01-15_astro_milkyway_001.jpg
2024-01-15_street_seattle_002.jpg
2015-06-20_wildlife_eagle_001.jpg

Why this works:
- Sorts chronologically automatically
- Easy to find by date or category
- Descriptive but not excessive
```

**For Scanned Film:**
```
Format: YYYY-MM-DD_film-type_subject_###.jpg

Examples:
2024-02-01_holga_portrait_001.jpg
2024-02-01_35mm-bw_street_002.jpg
1995-06-15_mediumformat_landscape_001.jpg (approximate date)

Notes:
- Use best-guess date for old film
- Film type helps track workflow
- Subject keeps searchable
```

### Batch Renaming Tools

**Windows:**
- Bulk Rename Utility (free)
- PowerToys (Microsoft, free)

**Mac:**
- Finder (built-in: Select all → Right-click → Rename)
- Name Changer (free)

**Cross-platform:**
- Adobe Bridge (batch rename)
- ExifTool (command line)

## 🧠 Mental Frameworks

### Dealing with Overwhelm

**When you feel stuck:**

1. **Use the 2-Minute Rule**
   - Can I upload just 1 image in 2 minutes?
   - Yes = do it now
   - Momentum builds momentum

2. **Batch by Emotion, Not Perfection**
   - Pick images that make you happy
   - Skip the ones that stress you out
   - Come back to "problem" images later

3. **Progress > Perfection**
   - A blurry upload is better than no upload
   - You can recategorize later
   - Tags can be added anytime

4. **Visible Progress = Motivation**
   - Check camera stats widget daily
   - Watch map fill with pins
   - See portfolio collections grow

### The "Just One" Technique

**Feeling resistant? Commit to uploading just ONE image.**

After ONE:
- Usually energized to do more
- Momentum started
- Day not wasted
- Count: 1 > 0

**ONE image per day = 365 per year**

## 📋 Decision-Making Shortcuts

### Category Decision (5 seconds max)

**Ask: "What's the PRIMARY subject?"**
- Landscape with a bird? → Choose one (Wildlife or Landscape)
- Street scene at night? → Street (time of day = tag)
- Portrait in nature? → Portraits (location = tag)

**Default categories:**
1. Astro
2. Wildlife
3. Street
4. Landscape
5. Portraits
6. Macro
7. Travel
8. Film/Holga

**Rule:** If unsure, pick "Travel" or create "Uncategorized" and fix later.

### Tag Decision (10 seconds max)

**Always add 2-3 tags minimum:**

**Format:**
1. Technique tag: holga, digital, film, black-and-white
2. Subject tag: dogs, urban, nature, people
3. Location tag: seattle, iceland, local, international

**Examples:**
- Milky Way photo: `astro, digital, iceland`
- Street portrait: `street, film, holga, seattle`
- Dog photo: `portraits, digital, commissioned, dogs`

**Rule:** Tags are easy to add later - don't overthink.

## 🎨 Workflow Optimizations

### Keyboard Shortcuts (Learn These)

**WordPress Media Library:**
- `Tab` → Next field
- `Enter` → Save
- `Shift + Click` → Select multiple
- `Ctrl/Cmd + Click` → Select individual

**Photoshop:**
- `Ctrl/Cmd + Shift + S` → Save As
- `Ctrl/Cmd + Alt + Shift + W` → Save for Web
- Create Action for "Export with Metadata"

**DxO PhotoLab:**
- Save export preset: "Web Upload - Keep Metadata"
- Assign to F-key for quick access

### Batch Processing Workflow

**For 20+ Images from Same Session:**

1. **Import to Lightroom/Bridge**
   - Apply basic edits to all
   - Star best 5-10

2. **Export Starred**
   - Use preset
   - Destination: ReadyToUpload folder

3. **Batch Upload to WordPress**
   - Media → Add New
   - Drag entire folder
   - Use bulk edit for category
   - Add tags individually (worth it)

4. **Quick Check**
   - View one in Media Library
   - Verify GPS/camera data present
   - If good, process next batch

## 🔄 The Scanning Workflow

### For Your Film Backlog

**Prioritization System:**

**Tier 1: Scan First (High Value)**
- Best/favorite rolls
- Commissioned work
- Irreplaceable moments
- Photos you'd print

**Tier 2: Scan Soon (Medium Value)**
- Complete rolls (easier to process)
- Specific projects/trips
- Quality images you like

**Tier 3: Scan Someday (Low Priority)**
- Test rolls
- Duplicates
- Uncertain quality
- Experimental work

**Scanning Schedule:**

**Weekly Session (2 hours, Sunday afternoon):**
1. Pull ONE roll from Tier 1
2. Scan at medium quality (save time)
3. Quick edit in DxO
4. Export to ReadyToUpload
5. Upload Mon-Fri using daily system

**Monthly Deep Scan (4 hours, first Saturday):**
1. Pull 3-5 rolls from Tier 1
2. Scan at high quality
3. Batch process
4. Create dedicated portfolio collection
5. Schedule uploads over next 2 weeks

**Rule:** Scan at "good enough" quality. Perfect scans take forever. Get them online first.

```batch
REM Auto-move today's exports to ReadyToUpload
xcopy "C:\DxO-Exports\*.jpg" "C:\PhotoArchive\ReadyToUpload\" /D /Y
```

**Mac Automator:**
1. Create workflow
2. Watch folder: DxO-Exports
3. Action: Copy to ReadyToUpload

```bash
# Add watermark text to all JPEGs
exiftool -Artist="Zachary Folk" -Copyright="© Zachary Folk 2024" *.jpg

exiftool "-FileName<CreateDate" -d "%Y-%m-%d_%%f.%%e" *.jpg
```

## 📞 When You're Stuck

### Common Sticking Points

**"I don't know which images to upload"**
→ Upload the first 5 you see. Seriously.

**"None are good enough"**
→ Lower your standards temporarily. Upload anyway.

**"I have 10,000 images to organize first"**
→ No, you don't. Start uploading now. Organization happens through uploading.

**"My metadata is messed up"**
→ Fix going forward. Don't let past stop present.

**"I'm weeks behind my goal"**
→ Reset the goal. Start from today.

**"This is taking too long"**
→ Reduce daily target. 1 image > 0 images.
