# 30-Day Photography Website Launch Plan

## 🎯 Overview

A structured, bite-sized approach to organizing thousands of photos and launching your photography website. **15-30 minutes per day** is all you need.

---

## Philosophy: Small Wins Daily

**The Problem:** Thousands of images = overwhelming paralysis
**The Solution:** Micro-tasks + daily progress + visible results

**Key Principles:**
1. **Progress over perfection** - Done is better than perfect
2. **Small batches** - Work in sets of 5-10 images
3. **Visible wins** - See results on your site quickly
4. **No backlog guilt** - Focus on moving forward

---

## Week 1: Foundation (Days 1-7)

### Day 1: Setup & Configuration (30 min)
**Goal:** Get the technical foundation ready

- [ ] Activate FolkPhotography theme
- [ ] Configure hero image category (Appearance → Customize)
- [ ] Upload 5 hero-worthy images to "Hero Images" category
- [ ] Test hero on homepage - does it show a random image?

**Success:** Homepage loads with rotating hero image

---

### Day 2: Create Core Categories (20 min)
**Goal:** Establish your organizational structure

- [ ] Go to Posts → Categories
- [ ] Create 5-8 main categories based on YOUR work:
  - Example: Astro, Wildlife, Street, Macro, Portraits, Travel, Film, Landscape
- [ ] Write 1-2 sentence descriptions for each
- [ ] Don't overthink it - you can adjust later

**Success:** Categories created and saved

---

### Day 3: Homepage Widgets Setup (25 min)
**Goal:** Make your homepage functional

- [ ] Appearance → Widgets
- [ ] **Homepage - After Hero:** Add "Recent Portfolio Items" (6 items, 3 columns)
- [ ] **Homepage - Featured:** Add "Camera & Photography Stats"
- [ ] **Homepage - Gallery:** Add "Category Gallery" widget (pick your strongest category)
- [ ] Save and view homepage

**Success:** Homepage has 3 active sections

---

### Day 4: First 5 Images Upload (30 min)
**Goal:** Get comfortable with the upload workflow

**Pick 5 of your BEST images** (don't overthink, just pick):
- [ ] Upload to Media Library
- [ ] For each image:
  - [ ] Add descriptive title (e.g., "Milky Way over Yosemite")
  - [ ] Assign 1 category
  - [ ] Add 2-3 tags (subject, technique, location)
- [ ] Check if GPS data was extracted (Location Data meta box)

**Success:** 5 images uploaded with metadata

---

### Day 5: First Blog Post (25 min)
**Goal:** Create your first content

- [ ] Posts → Add New
- [ ] Title: "Welcome" or "About My Photography"
- [ ] Write 3-4 paragraphs about your work
- [ ] Add 2-3 of your uploaded images
- [ ] Assign category: "Behind the Scenes" or "Journal"
- [ ] Publish

**Success:** First post live on site

---

### Day 6: First Portfolio Collection (30 min)
**Goal:** Create your first curated collection

- [ ] Portfolio → Add New
- [ ] Title: Your best photo series (e.g., "Iceland 2024")
- [ ] Add 6-10 images as gallery
- [ ] Write 2-3 paragraph description
- [ ] Assign portfolio category
- [ ] Set featured image
- [ ] Publish

**Success:** First portfolio item live

---

### Day 7: Week Review + Masonry Page (20 min)
**Goal:** Create portfolio showcase page

- [ ] Review what you've accomplished (you're 7 days in!)
- [ ] Pages → Add New
- [ ] Title: "Portfolio" or "Gallery"
- [ ] Select "Masonry Gallery" template
- [ ] Add intro text (2-3 sentences)
- [ ] Publish
- [ ] Add to menu (Appearance → Menus)

**Success:** Portfolio page live and linked in menu

## Week 2: Building Momentum (Days 8-14)

### Daily Pattern: "The Rule of 5"

**Every day, do ONE of these tasks (15-20 min):**

**Option A: Add 5 New Images**
- Upload 5 photos
- Title, category, 2-3 tags each
- From same shoot or theme for efficiency

**Option B: Create 1 Portfolio Item**
- Pick a theme/location/series
- Add 6-10 images
- Write description
- Publish

**Option C: Write 1 Blog Post**
- Short photo story
- Behind-the-scenes
- Technique explanation
- 3-5 images

**Option D: Organize Existing**
- Pick 5 images already uploaded
- Improve titles/tags
- Add to portfolio collection
- Update descriptions

**Track Your Progress:**
```
Day 8:  ☐ Added 5 images (Street Photography)
Day 9:  ☐ Created Portfolio: "Dogs of San Francisco"
Day 10: ☐ Blog post: "My Holga Journey"
Day 11: ☐ Added 5 images (Travel - Iceland)
Day 12: ☐ Organized existing images
Day 13: ☐ Added 5 images (Astro)
Day 14: ☐ Week 2 Review - Count total images uploaded
```

**End of Week 2 Goals:**
- At least 30 images on site
- 2-3 portfolio collections
- 2-3 blog posts
- Portfolio page looking good

## Week 3: System Refinement (Days 15-21)

### New Element: Batch Processing

**Goal:** Develop efficient workflows for processing scattered images

### Day 15: Create Folder Structure (30 min)
**Set up a staging system on your computer**

```
C:\PhotoStaging\
├── ToScan\           # Film needing scanning
├── ToProcess\        # Digital RAW files
├── ReadyForWeb\      # Processed, ready to upload
└── Uploaded\         # Archive after upload
    ├── 2024-02\
    └── 2024-01\
```

- [ ] Create these folders
- [ ] Move 20-30 photos to appropriate folders
- [ ] This is your NEW workflow staging area

---

### Day 16-21: Themed Batch Days

**Each day = one theme** (20-30 min per day)

**Monday: Travel Day**
- Process 5-10 travel images
- Upload with location tags
- Check GPS extraction

**Tuesday: Portrait Day**  
- Process 5-10 portrait images
- Tag subjects appropriately
- Create portfolio if you have a series

**Wednesday: Film Scan Day**
- Scan 5 film frames
- Process in DxO/Photoshop (see metadata preservation below)
- Upload immediately

**Thursday: Astro/Night Day**
- Process 5-10 night sky images
- Technical tags (ISO, exposure time)

**Friday: Project Day**
- Work on portfolio collection
- OR write blog post
- OR improve existing content

**Weekend: Flex Days**
- Catch up if you missed days
- OR take a break
- OR work ahead

## Breaking Through Overwhelm

### When You Feel Stuck

**Feeling: "I have too many photos to organize"**
- **Action:** Process just 5 images today. That's it.
- **Reminder:** You don't need to upload everything. Feature your BEST work.

**Feeling: "My old photos aren't categorized"**
- **Action:** Stop. Focus on NEW uploads only.
- **Reminder:** Backfilling old content is a future project.

**Feeling: "I don't know which images to upload"**
- **Action:** Pick 5 from your most recent shoot.
- **Reminder:** Don't curate yourself to death. Upload, see what works.

**Feeling: "Scanning film is overwhelming"**
- **Action:** Scan 1 roll per week. Just one.
- **Reminder:** Film can wait. Focus on accessible digital first.

**Feeling: "I need to process everything perfectly"**
- **Action:** "Good enough" processing + uploaded > Perfect processing + never uploaded
- **Reminder:** You can replace images later.

### The "5-Minute Rule"

**Too tired/busy/unmotivated?**

Commit to just 5 minutes:
1. Open WordPress
2. Upload 1 image
3. Add title and category
4. Save

Usually you'll continue. If not, 1 image is still progress.

## Batch Processing Strategies

### For Digital RAW Files (Scattered Locations)

**Monthly Consolidation Day** (1 hour)
1. Search computer for all RAW files
2. Copy to `C:\PhotoStaging\ToProcess\`
3. Sort by date/subject
4. Pick best 20-30 for this month
5. Process in batches

### For Film Scans

**Weekly Scan Day** (1 hour)
1. Pick 1 roll to scan
2. Scan all frames (even bad ones)
3. Quick review - flag best 5-10
4. Process flagged images
5. Upload immediately
6. Archive roll to "Scanned" folder

### For Already-Processed JPGs

**Quick Upload Day** (30 min)
1. Find folder of old JPGs
2. Pick best 10 images
3. Batch rename with descriptive names
4. Upload all 10 at once
5. Assign category in bulk (WordPress bulk edit)
6. Add individual tags later (or not at all)

## Photography-Specific Tips

### Dealing with Film Backlog

**Reality Check:** You probably have 100+ rolls to scan.

**Strategy: The 20% Rule**
- 20% of your frames are worth scanning well
- Scan everything QUICK on first pass (low res, auto settings)
- Review, flag best 20%
- Re-scan flagged images at high quality
- Upload only the re-scanned images

### Digital RAW Workflow

**Batch Processing in DxO/Photoshop:**
1. Process 10-15 images with similar settings
2. Export all to `ReadyForWeb` folder
3. Upload in one batch
4. Bulk edit category in WordPress
5. Add tags individually (5 min)
